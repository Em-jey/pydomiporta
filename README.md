# Running

### Installation
Version from https://pypi.python.org/pypi/pydomiporta:
```
pip install pydomiporta
```
Latest version from git:
```
pip install git+git://github.com/limebrains/pydomiporta.git
```

### Example script
```
python example.py
```

### Travis pipeline
```
tox
```

### Tests
```
py.test tests.py -vv --pdb

```
